### Librerie usate:
  - `pygame`, versione: `2.6.0` ( **NOTA**: dipendenza da `numpy` )
 
### Comandi gioco:
   - `esc` per tornare al menù (comando globale)
   - `freccia su` per saltare
   - `freccia giù` per abbassarsi
   - `invio` per confermare il nome e successivamente giocare
